# Geospatial-Analysis-Project

This project analyzes regional sales data and identifies potential expansion areas using Python.

---

## Features

- Visualizes sales data using a bar chart
- Identifies high-demand regions without store presence
- Creates an interactive map with location markers
- Highlights expansion areas on the map

---

## Technologies Used

- Python 
- Pandas (Data Analysis)
- Matplotlib (Graph Visualization)
- Folium (Interactive Maps)

---

## Project Files

- `geospatial_analysis.py` → Main Python script  
- `sales_location_data.csv` → Dataset  
- `graph.png` → Sales graph  
- `map.html` → Interactive map  

---

## Output

### Sales Graph
Displays sales by region using a bar chart.

### Interactive Map
- Markers show all regions  
- Red circles highlight expansion areas (high demand + no store)
